Installing Karamelo on Ubuntu:

1. Install all the dependencies shipped with ubuntu:
sudo apt install make gcc g++ libz-dev cmake libopenmpi-dev libeigen3-dev

2. Add Eigen to your paths:

Add to you .bash_profile:

CPATH=$CPATH:/usr/include/eigen3
export CPATH

LIBRARY_PATH=$LIBRARY_PATH:/usr/include/eigen3
export LIBRARY_PATH

3. Download and install Karamelo:

git clone https://github.com/adevaucorbeil/karamelo.git
cd ~/karamelo
cmake -DCMAKE_BUILD_TYPE=release build .
make

4. Enjoy!

Installing Karamelo on macOS:

1. Install all the dependencies:
brew install make gcc cmake eigen open-mpi libomp

2. Download and install Karamelo:

git clone https://github.com/adevaucorbeil/karamelo.git
cd ~/karamelo
cmake -DCMAKE_BUILD_TYPE=release build .
make

3. Enjoy!
