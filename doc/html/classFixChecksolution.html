FixChecksolution Class Reference
fix

Detailed Description

Syntax

fix(fix-ID, check_solution, group-ID, ux, uy, uz)
  • fix-ID: name of the fix to be created.
  • fix-type: velocity_nodes, force_nodes, kinetic_energy, ...
  • group-ID: name of the group onto which the fix will be applied. If 'all' is used, all particles will be selected.
  • ux, uy and uz: solution of the particle's displacements in the x, y and z directions.

Examples

c = sqrt(E/rho)
G = 0.05
phi2 = PI
phi3 = PI
ux = G*sin(PI*x)*sin(c*PI*time)
uy = G*sin(PI*y)*sin(c*PI*time + phi2)
uz = 0
fix(error, check_solution, all, ux, uy, uz)

This fix calculates the error between all the particles' current position and their theoretical position given by \(\boldsymbol{u}=[u_x(x,t), u_y(y,t), u_z]\). The resulting error can be accessed using the error_s, error_x, error_y and error_z variables.

Description

This command determine the error in displacement of the particles from the group 'group-ID', being provided the solution \(\boldsymbol{u}=[ux, uy, uz]\), where ux, uy and uz are functions of positions and time. The fix generates four variables:

  • fix-ID_s:

    \begin{equation} e(t^n) = \sqrt{\frac{\sum_{p=0}^{N_p} V_p^0\left|\left|\boldsymbol{u}_p^h(t^n) - \boldsymbol{u}(\boldsymbol{X}_p, t^n)\right|\right|^2}{V_{tot}^0}} \end{equation}

  • fix-ID_x:

    \begin{equation} e = \sum_{t^n=0}^{T} \sum_{p=0}^{N_p} V_p^0\left|\left|\boldsymbol{u}_p^h(t^n) - \boldsymbol{u}(\boldsymbol{X}_p, t^n)\right|\right|^2 \end{equation}

  • fix-ID_y:

    \begin{equation} e = \sum_{t^n=0}^{T} \sum_{p=0}^{N_p} V_p^0\left|\left|\boldsymbol{u}(\boldsymbol{X}_p, t^n)\right|\right|^2 \end{equation}

  • fix-ID_z:

    \begin{equation} e_2 = \sqrt{\frac{\sum_{t^n=0}^{T} \sum_{p=0}^{N_p} V_p^0\left|\left|\boldsymbol{u}_p^h(t^n) - \boldsymbol{u}(\boldsymbol{X}_p, t^n)\right|\right|^2}{\sum_{t^n=0}^{T} \sum_{p=0}^{N_p} V_p^0\left|\left|\boldsymbol{u}(\boldsymbol{X}_p, t^n)\right|\right|^2}} \end{equation}
