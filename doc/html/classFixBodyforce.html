FixBodyforce Class Reference
fix

Detailed Description

Syntax

fix(fix-ID, body_force, group-ID, fx, fy, fz)
  • fix-ID: name of the fix to be created.
  • fix-type: velocity_nodes, force_nodes, kinetic_energy, ...
  • group-ID: name of the group onto which the fix will be applied. If 'all' is used, all particles will be selected.
  • fx, fy and fz: x, y and z components of the body force to be applied, respectively.

Examples

fix(fbody, body_force, all, 0, -9.81, 0)

Applies a body force \(\boldsymbol{f}_b = [0, -9.81, 0]\) onto all particles in the simulation box.

Description

This command defines a body force \(\boldsymbol{f}_b\) that is applied to all the particles in the group 'group-ID'non-linear equation of state:

\begin{equation} \boldsymbol{f}_b = \begin{bmatrix} fx\\ fy\\ fz \end{bmatrix} \end{equation}
