eos

Detailed Description

Syntax

eos(eos-ID, fluid, rho0, K, gamma)
  • eos-ID: name of the eos to be created.
  • rho0: reference bulk density
  • K: bulk modulus.
  • gamma: Tait exponent.

Examples

eos(eosf, fluid, rho, K, gamma)

Defines a non-linear EOS called 'eosf' typically used for the simulation of fluids in MPM with a reference bulk density rho, a bulk modulus K and the Tait coefficient gamma.
